# mdserve 简介

mdserve 是一个实时 Markdown 文件服务器，它提供了优雅的 Web 界面来浏览和渲染 Markdown 文件。

## 什么是 mdserve？

mdserve 让你可以：

- 📁 **组织文档** - 按目录结构管理你的 Markdown 文件
- 🔍 **搜索内容** - 快速搜索文件名和文件内容
- 🌓 **切换主题** - 支持亮色和暗色主题
- 📑 **目录导航** - 自动生成文档大纲
- 🏷️ **标签分类** - 使用标签和分类组织文档
- ⚡ **实时预览** - 文件修改后自动刷新

## 适用场景

mdserve 适用于：

| 场景 | 描述 |
|------|------|
| 📚 文档站 | 项目文档、API 文档 |
| 📝 知识库 | 个人笔记、团队知识库 |
| 📖 博客 | 技术博客、教程网站 |
| 📋 手册 | 操作手册、用户指南 |

## 为什么选择 mdserve？

1. **简单易用** - 单个命令即可启动
2. **零配置** - 开箱即用
3. **单文件部署** - 前端资源嵌入二进制
4. **跨平台** - 支持 Windows、macOS、Linux
