# 示例文档

欢迎使用 mdserve 示例文档！这个目录包含了展示 mdserve 各种功能的示例文件。

## 目录结构

```
example/
├── README.md                    # 本文件
├── getting-started/
│   ├── introduction.md          # 快速入门介绍
│   └── installation.md          # 安装指南
├── guides/
│   ├── markdown-basics.md       # Markdown 基础语法
│   ├── front-matter.md          # Front Matter 元数据
│   └── advanced-formatting.md   # 高级格式化
└── tutorials/
    ├── tutorial-1.md            # 教程 1
    ├── tutorial-2.md            # 教程 2
    └── tutorial-3.md            # 教程 3
```

## 功能演示

### 标签和分类

每个文档都可以包含标签和分类，通过 YAML Front Matter 定义：

```yaml
---
title: "文档标题"
tags:
  - tag1
  - tag2
categories:
  - 分类1
---
```

### 主题切换

点击右上角的主题切换按钮，可以在亮色和暗色主题之间切换。

### 目录导航

右侧边栏会自动显示文档的目录大纲，方便快速导航。

### 实时刷新

当文件被修改时，浏览器会自动刷新显示最新内容。

## 开始探索

点击左侧的文件树，开始探索这些示例文档！
