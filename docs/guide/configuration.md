# 配置选项

mdserve 支持多种配置选项。

## 命令行参数

| 参数 | 简写 | 默认值 | 说明 |
|------|------|--------|------|
| `--port` | `-p` | 3000 | 服务器监听端口 |
| `--host` | | 127.0.0.1 | 服务器监听地址 |

## 示例

### 本地开发

```bash
mdserve serve ./docs
```

### 局域网访问

```bash
mdserve serve ./docs --host 0.0.0.0
```

### 自定义端口

```bash
mdserve serve ./docs -p 8080
```

### 生产环境

```bash
mdserve serve ./docs --host 0.0.0.0 --port 80
```

## 注意事项

- 默认只监听本地回环地址 (127.0.0.1)
- 如需外部访问，请设置 `--host 0.0.0.0`
- 确保防火墙开放相应端口
